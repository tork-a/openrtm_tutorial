1   Related Tutorials¶

You’ll see very basic operation for OpenHRI, SEATSAT. See the following links for more.

  • Detail of the demo in this tutorial (in Japanese). Follow ‘Running the system’, ‘Running example’ at the lower side in the page [1]

    • Same software is bundled in ./demo/0_SpeechDemo in the USB.

    • Same PDF files in bundled at ./doc/tutorial_openhri_mediaplay.pdf

2   HowToRun¶

Multiple windows get opened and overlap with each other. Find appropriate one.

  1. Click in the following order “0_StartDemo.bat”，”1_ConnectRTC.bat”，”2_ActivateRTC.bat” If they don’t seem to be working, on Explorer open ”./demo/0 SpeechDemo” and run them.(Since this demo uses Japanese language processor, we’re not sure if this works in English though)

  • ./demo/0_SpeechDemo/0_StartDemo.bat
  • ./demo/0_SpeechDemo/1_ConnectRTC.bat
  • ./demo/0_SpeechDemo/2_ActivateRTC.bat

  1. Move the window “tk” forward.

  2. Say “Video”. A window whose title is “video_list_mode” appears.

  3. Say “list” will open a window “PlayList”.

  4. (Not sure if this works in English though) Say “x ban-me” (x = 1...5) will pop up a window “PlayList” where highlighted item changes per the number you say.

  5. Say “Play” will play the content on a window “Player”

  6. Say “Stop” and other commands that are available on the GUI.

  7. When you finish the demo, click “3_DeactivateRTC.bat”, “4_DisconnectRTC.bat”, “5_DemoExit.bat” respectively

  • ./demo/0_SpeechDemo/3_DeactivateRTC.bat
  • ./demo/0_SpeechDemo/4_DisconnectRTC.bat
  • ./demo/0_SpeechDemo/5_DemoExit.bat

Deactivation of RTCs might take awhile depending on the amount of the resource loaded by each component.
