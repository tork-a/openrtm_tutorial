1   What you’ll experience¶

  • Play motion via voice on Choreonoid

2   Related Tutorials¶

You’ll see very basic operation for Choreonoid and related RTC in this tutorial. See the following links for more details.

  • Sample demo to operate robot’s arm via voice on `Choreonoid`[3]

  • RobotMotionRtc – sample RTC to operate robot via OpenRTM on `Choreonoid`[1]

  • Startup guide for Choreonoid by using `G-ROBOTS GR-001`[2]

2.1   SystemEnvironment¶

  • Windows 7／ 8

  • A microphone recognized by Windows

3   HowToRun¶

Your current directory is assumed to be the home of the USB unless specified.

3.1   Run Common Tools¶

In case they are not running.

  • CORBA Nameserver (rtm-naming.bat)
  • ./demo/RTSE.bat

3.2   Run This Tutorial¶

  1. Click “0_StartDemo.bat”，”1_ConnectRTC.bat”，”2_ActivateRTC.bat” respectively.If it doesn’t work, run it manually in ”./demo/0 SpeechDemo” from Explorer

  • ./demo/MotionByVoiceDemo/0_StartDemo.bat
  • ./demo/MotionByVoiceDemo/1_ConnectRTC.bat
  • ./demo/MotionByVoiceDemo/2_ActivateRTC.bat

  1. (Optional) To debug, you can run 0 to 2 .bat files in ./demo/SampleWordLogger (an RTC that records what’s spoken).

Show the connection status of RTCs that are run in the .bat files on RT System Editor (If you didn’t run SampleWordLogger, you just don’t see SampleWordLogger0 RTC):

  1. Say toward microphone “migi a-ge-te” (meaning: raise right hand, halt for about half an second at whitespace is better)

  2. Other possible combinations:

right | left

+

a-ge-te|sa-ge-te|a-ge-nai|sa-ge-nai

  1. Say “sai-sei yoro-shiku” (meaning: let’s play) will let the robot move according to SampleMotion1.

  2. Click “3_DeactivateRTC.bat”, “4_DisconnectRTC.bat”，”5_DemoExit.bat”, respectively.

  • ./demo/MotionByVoiceDemo/3_DeactivateRTC.bat
  • ./demo/MotionByVoiceDemo/4_DisconnectRTC.bat
  • ./demo/MotionByVoiceDemo/5_DemoExit.bat

4   Managed status of the robot¶

(Though this is the topic in the next tutorial[4]) the demo here uses a module called SEAT for managing status. The statuses handled in this demo are, as defined in ./demo/MotionByVoiceDemo/motion_cnoid.seatml, the combination of arms positions such as both_down, both_up, left_up, right_up.
